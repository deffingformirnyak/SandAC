# SandAC (Sand Anti-Cheat)

[English](#english) | [Русский](#русский)

---

## English

**SandAC** is a lightweight and efficient anti-cheat solution designed for Minecraft servers (Spigot/Paper). It focuses on detecting common cheats and providing moderation tools to ensure a fair gaming environment.

### Key Features:
*   **Combat Checks:** Detection of KillAura, Reach, and other combat-related hacks.
*   **Movement Checks:** Prevention of Fly, Speed, Step, and various movement exploits.
*   **Bait Bots:** Advanced detection using invisible "bait" entities to catch KillAura users.
*   **Packet Analysis:** Deep packet inspection to identify malicious client behavior.
*   **Moderation System:** Built-in commands for muting, banning, and punishing rule-breakers.
*   **Alert System:** Real-time notifications for staff members when a violation is detected.
*   **Performance Optimized:** Designed to have minimal impact on server performance.

## Русский

**SandAC** — это легковесное и эффективное решение для защиты от читов, разработанное для серверов Minecraft (Spigot/Paper). Проект сфокусирован на обнаружении популярных читов и предоставлении инструментов модерации для обеспечения честной игры.

### Основные функции:
*   **Проверки боя (Combat):** Обнаружение KillAura, Reach и других боевых хаков.
*   **Проверки перемещения (Movement):** Предотвращение Fly, Speed, Step и различных эксплойтов движения.
*   **Боты-приманки (Bait Bots):** Продвинутое обнаружение с использованием невидимых сущностей для поимки пользователей KillAura.
*   **Анализ пакетов:** Глубокая проверка пакетов для выявления вредоносного поведения клиента.
*   **Система модерации:** Встроенные команды для мута, бана и наказания нарушителей.
*   **Система оповещений:** Уведомления персонала в реальном времени при обнаружении нарушений.
*   **Оптимизация производительности:** Минимальное влияние на работу сервера.
