# SandAC Update Log (v1.01)

[English](#english) | [Русский](#русский)

---

## English

### Recent Changes:
*   **Temporary Removal:** The ElytraFlight check was temporarily disabled due to potential bugs and errors; it will be re-enabled after optimization.
*   **Configurable Permissions:** All command permissions (alerts, ban, mute, reload, etc.) can now be customized in `config.yml`.
*   **New Reload Command:** Added `/sandac reload` to apply configuration changes instantly without restarting the server.
*   **Punishment System Overhaul:**
    *   Added a toggle to enable or disable automatic punishments (`punishment.enabled`).
    *   Added configurable ban duration, supporting both milliseconds and `"permanent"`.
    *   Added customizable ban reasons and kick messages with `{check}` placeholder support.
*   **Code Cleanup:** Removed all comments from the Java source files for a cleaner and more streamlined codebase.

## Русский

### Последние изменения:
*   **Возможные ошибки по поводу:** Чека на ElytraFlight, и временно была удалена!
*   **Настраиваемые права:** Все права на команды (alerts, ban, mute, reload и т.д.) теперь можно настроить в `config.yml`.
*   **Новая команда перезагрузки:** Добавлена команда `/sandac reload` для мгновенного применения изменений конфигурации без перезагрузки сервера.
*   **Обновление системы наказаний:**
    *   Добавлен переключатель для включения или выключения автоматических наказаний (`punishment.enabled`).
    *   Добавлена настраиваемая длительность бана, поддерживающая как миллисекунды, так и значение `"permanent"`.
    *   Добавлены настраиваемые причины бана и сообщения об исключении с поддержкой плейсхолдера `{check}`.
*   **Очистка кода:** Из всех исходных файлов Java удалены комментарии для обеспечения чистоты и лаконичности кодовой базы.
